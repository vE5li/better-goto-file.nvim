# Installation

Lazy:

```lua
return {
    "ve5li/better-goto-file.nvim",
    config = true,
    ---@module "better-goto-file"
    ---@type better-goto-file.Options
    opts = {},
}
```

# Usage

Invoke via the `GotoFile` command or by calling `require("better-goto-file").goto_file(opts)` from Lua.
